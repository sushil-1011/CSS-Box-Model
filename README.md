# CSS Box Model 

## Project Overview

This repository contains practice files created to understand the CSS Box Model and related styling concepts in Full Stack Development (FSD).

The project demonstrates how elements are structured and styled using:

- Margin
- Border
- Padding
- Content
- Colors
- Fonts and Text properties

All files are created using HTML and CSS for learning purposes.


## Objectives

- Understand the CSS Box Model concept
- Learn how margin, border, padding, and content work
- Practice color styling in CSS
- Learn font and text properties
- Improve overall webpage design skills


## Technologies Used

- HTML5
- CSS3

## File Description

### 1. Box_model.html

Demonstrates the CSS Box Model structure:
- Content
- Padding
- Border
- Margin

Shows how spacing affects layout and element size.


### 2. Color_Reference.html

Demonstrates different ways to apply colors in CSS:
- Color names
- HEX values
- RGB values
- Background colors
- Text colors


### 3. Document.html

Demonstrates basic HTML document structure and styling.
Shows how CSS can be applied to structure content properly.


### 4. Font and text .html

Demonstrates font and text properties such as:
- Font family
- Font size
- Font weight
- Text alignment
- Text decoration
- Line height


### 5. messi.jpg

Image file used in the project to demonstrate styling, spacing, and layout with images.


## How to Run

1. Clone the repository:

```
git clone <repository-link>
cd CSS-Box-Model
```

2. Open any HTML file in a web browser.

No additional installation or dependencies are required.
